# React + Vite

A premium, responsive weather dashboard built with React, Vite, and OpenWeather API.

Features
Real-time Weather: Fetch current weather data for any city.
5-Day Forecast: View daily weather trends.
Dynamic UI: Glassmorphism design with responsive layouts.
Error Handling: Friendly messages for invalid cities or API issues.
Lucide Icons: High-quality SVG icons for weather details.
Tech Stack
React: Frontend framework.
Vite: Fast development build tool.
Axios: API requests.
Lucide React: Icon library.
Vanilla CSS: Custom premium styling with CSS variables.
